# EduTrack — Premium AI Educational Ecosystem

EduTrack is an enterprise-grade platform designed to revolutionize the way students learn, teachers collaborate, and creators monetize educational content.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL
- NPM or PNPM

### Installation

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd EduTrack
   ```

2. Install dependencies for both frontend and backend:
   ```bash
   # Frontend
   cd frontend
   npm install

   # Backend
   cd ../backend
   npm install
   ```

3. Setup Environment Variables:
   - Create a `.env` file in `frontend` and `backend` using the provided `.env.example`.

4. Run the development servers:
   ```bash
   # Frontend
   npm run dev

   # Backend
   npm run start:dev
   ```

## 🛠️ Tech Stack
- **Frontend**: Next.js 16, Tailwind CSS 4, Framer Motion, Lucide Icons.
- **Backend**: NestJS, Prisma ORM, PostgreSQL.
- **AI**: OpenAI GPT-4o Integration.
- **Real-time**: Socket.IO.

## 📁 Project Structure
- `frontend/`: Next.js application with App Router.
- `backend/`: NestJS server with modular architecture.
- `prisma/`: Database schema and migrations.

## 💎 Key Features
- **EduAI**: Advanced AI study assistant.
- **Multi-Institution**: Isolated environments for colleges and schools.
- **Video Platform**: Educational creator ecosystem with monetization.
- **Chat**: Discord-inspired real-time communication.
- **Productivity**: AI-powered task management and study streaks.
